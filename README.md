# Redux Toolkit Todo App

A modern Todo application built with React and Redux Toolkit that demonstrates centralized state management, CRUD operations, and efficient state updates using Redux Toolkit best practices.

---

# 🚀 Features

* Add Todos
* Edit Todos
* Delete Todos
* Mark Todos as Completed
* Filter Completed & Pending Tasks *(if implemented)*
* Persistent State *(if implemented using localStorage)*
* Responsive User Interface

---

# 🔄 State Management Flow

1. User performs an action (Add/Edit/Delete Todo)
2. Component dispatches a Redux action
3. Reducer updates the application state
4. Redux Store stores the updated state
5. React automatically re-renders affected components
